# 器成天下走

> 一件瓷器的诞生有多难？当手艺面对御命与期限，匠人又要付出什么？

《器成天下走》是一款以古代景德镇瓷业为背景的网页经营策略游戏。玩家经营一座窑坊，在高岭山、松柴林、瓷作坊、窑炉和昌江码头之间安排生产，经历从采土、制坯、绘画到烧成和交付的完整流程。

## 核心体验

- **完整制瓷链条**：采集高岭土、松柴、釉料和颜料，完成制坯、绘画、烧窑与交付。
- **资源与体力经营**：每项劳作都会消耗体力，需要在原料、库存、订单期限和工钱之间取舍。
- **四季与窑火**：季节影响生产效率、柴火消耗和烧制品质；升级窑炉可以提高生产能力。
- **市集与御命**：普通订单带来银两和声望；第三单后会触发一次限时御命，要求4件成瓷与16品质。
- **三档难度**：按钮会显示当前状态——`难度：易 / 中 / 难`。
- **拼合青花残片**：随时进入九宫格拼图，依靠纹样走向和编号复原瓷片。
- **场景科普旁注**：鼠标悬停或键盘聚焦场景即可预览对应工艺知识，无需先消耗资源；点击按钮才会执行劳作。
- **每日见闻**：度过一日时，30% 概率出现趣味彩蛋，30% 概率出现真实历史故事，40% 平静度过。
- **木刻叙事事件**：御命、童宾祭窑、秋后问斩等关键时刻以粗粝木刻画面表现权力与个体的冲突。
- **中后期窑务事件**：完成 8 笔订单后进入带选择、冷却与连续后果的事件牌库，选择会影响未来数日的柴耗、烧成率、售价、工钱、体力或码头交付。
- **一日排产**：完成 8 笔订单后可在匠作总台自动补缺、制坯、彩绘、批烧与交付；完全沿用原有资源和体力成本，遇到瓶颈会立即停止并说明原因。
- **四级窑炉**：从竖穴窑、龙窑、葫芦窑逐步升级到镇窑，地图名称、烧制按钮、成功率和器物品类同步变化。
- **运输与工艺成长**：订单会解锁更大的昌江船队并带来运输溢价；成功烧成会推动工艺从新手成长为天工。
- **三大行帮**：首次加入免费，之后可付40银改换。徽帮强化采集、都帮强化烧窑、杂帮强化彩绘品质。
- **后期匠作总台**：完成首单后解锁集中工序牌，采集、制坯、彩绘、烧制和交付无需再反复拖动地图。
- **可拖动工序牌**：总台只保留生产操作，按住标题可在屏幕内拖动，支持收起/展开并自动记住位置；商人与度日仍使用订单栏按钮。
- **可见的强征门槛**：强征按钮始终显示，皇恩不足10时直接标出解锁条件，达到门槛后恢复劳役与民怨机制。
- **窑工身份成长**：玩家从窑工、熟手、作头成长为把桩师傅；扩充窑班改为请督造官调人，职责关系更符合故事背景。
- **皇恩自然成长**：普通订单固定增加皇恩，彩绘与领先工艺追加奖励；完成御命增加20皇恩，商誉每达到25点也会触发一次入京里程碑。
- **三件批量烧制**：三件瓷坯分别独立判定成功率，一次消耗20体力并节省1捆松柴，结算页汇总烧成与损毁数量。
- **真实藏品拼图**：使用大都会艺术博物馆公共领域明代青花瓷瓶图像，改为无数字提示的滑块修复玩法。

## 按钮说明

- **采高岭土 / 伐松柴 / 采釉料 / 采颜料**：取得对应原料，每次消耗5体力；冬季伐柴额外消耗体力。
- **制作坯体**：消耗3份瓷土、1份釉料和8体力；匠人达到3人时通常一次制成2件。
- **彩绘装饰**：消耗1件素坯、1份颜料和4体力；烧成后增加6品质，订单中每件彩绘瓷另有25%装饰溢价。
- **入窑烧制**：消耗坯、松柴和12体力；按钮会显示当前窑炉与成功率。
- **交付订单**：消耗成瓷，获得银两、声望和通关进度，码头、工艺、商誉等加成在此结算。
- **请督造官调人**：玩家不直接行使招募权，而是花银请督造官向窑班调人；窑班人数会提高烧窑成功率，并满足制坯双产出条件，但每日工钱也会上升。
- **升级窑炉**：按竖穴窑→龙窑→葫芦窑→镇窑升级，提高基础烧成率。
- **度过一日**：恢复体力，推进季节、税收、工钱和订单期限，并有概率出现彩蛋或史话。
- **请道士**：花银为下一次烧窑增加临时成功率。
- **难度**：在易、中、难之间切换，影响税率、体力、雇佣/升级费用和通关单数。
- **商人**：买卖原料；购买不再降低商誉。商誉越高，进货越便宜、回售价越高、商人来访和订单溢价越好。
- **行帮**：选择一项长期专精；库存面板和按钮会显示当前所属行帮。
- **强征**：以皇恩推动生产，但会积累民怨；连续5次会导致民变毁窑。
- **重开**：清除当前存档并重新开始。
- **成就**：查看一次性挑战与银两奖励。
- **拼瓷片**：随时打开九宫格修复小游戏，完成后获得小奖励与修复知识。

## 美术方向

日常经营采用“古拙手稿风”，像翻阅一本边角磨损的窑工笔记；关键历史事件切换为黑白木版画，以更强烈的视觉语言表现御窑制度下的催逼、风险和牺牲。

地图使用横向长卷构图，从高岭山与松柴林延伸到作坊、龙窑和昌江码头。背景取法古代工艺图谱与木刻排线，瓷器的青花蓝则作为克制的视觉重点。

## 基本操作

1. 在地图上采集高岭土、松柴、釉料和颜料。
2. 制作素坯，并选择是否绘制彩绘坯。
3. 准备足够松柴和体力后入窑烧制。
4. 接取并交付订单，积累银两、声望和品质。
5. 度过一日恢复体力并结算税收、工钱、期限和随机见闻。
6. 招募匠人、升级窑炉，根据季节与订单调整生产节奏。

地图可按住空白处左右拖动，也可使用顶部的矿山、坯房、镇窑和码头定位按钮。

## 本地运行

项目是纯静态 HTML、CSS 和 JavaScript，无需安装依赖。

最简单的方式是直接打开 `index.html`。也可以在项目目录启动静态服务器：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000/`。

游戏存档保存在当前浏览器的本地存储中。更换浏览器、清理网站数据或使用不同域名时，存档不会自动迁移。

## 设计目标

游戏希望让玩家在经营决策中理解瓷器并非一件凭空出现的商品，而是泥土、燃料、经验、运输和无数次失败共同形成的结果。它既关注工匠精神，也关注御窑体系中器物留名、匠人无名的历史处境。

## 当前状态

项目仍处于可持续迭代的网页版原型阶段。后续可继续扩展真实瓷器图鉴、历史人物、贸易路线、窑口差异、更多残片修复关卡以及更完整的 RTS 式人员调度。

当前平衡版本中，普通订单需求最高增长到5件；简单、普通、困难分别需要完成5、8、12笔订单。通关或触发民变、秋后问斩等失败条件时，会进入独立的家谱结算页。
