器成天下走 · 瓷业经营

最新版项目说明、在线试玩和操作指南请查看 README.md。
在线地址：https://geniuswang23.github.io/qicheng-tianxia/

一款以古代景德镇瓷业为背景的轻经营策略网页游戏。

游玩方式
  双击"器成天下走.exe"启动，或运行 python launcher.py
  程序自动在浏览器中打开游戏页面

游戏特色
  - 完整制瓷产业链：采集高岭土、松柴、釉料 → 制坯 → 绘画装饰 → 烧制 → 交付订单
  - 四季系统：春夏秋冬各有不同效果，影响生产策略
  - 体力值管理：每个劳作消耗体力，合理安排每日工作
  - 烧制成功率：窑炉等级、匠人数量和季节共同影响，非100%成功
  - 成就系统：8 个成就等待解锁，达成奖励银两
  - 难度选择：简单/普通/困难，税率和通关条件不同
  - 税收系统：每 5 日官府课税，贴合历史背景
  - 突发事件：随机触发，考验应变能力
  - 御命订单：限时高品质挑战
